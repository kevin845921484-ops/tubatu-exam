# 📋 飞书多维表格配置指南

## 第一步：创建飞书多维表格

1. 访问 https://feishu.cn 并登录
2. 点击「云文档」→「新建」→「多维表格」
3. 选择「新建空白多维表格」

## 第二步：配置数据表结构

在新建的多维表格中，添加以下字段：

| 字段名称 | 字段类型 | 说明 |
|----------|----------|------|
| 姓名 | 文本 | 考生姓名 |
| 部门 | 文本 | 考生部门 |
| 工号 | 文本 | 考生工号 |
| 考试时间 | 日期时间 | 考试提交时间 |
| 单选题正确 | 数字 | 单选题正确题数 |
| 判断题正确 | 数字 | 判断题正确题数 |
| 论述题得分 | 数字 | 论述题得分 |
| 总分 | 数字 | 考试总分 |
| 结果 | 单选 | 优秀/合格/不合格 |

## 第三步：获取配置信息

### 3.1 获取 App Token

1. 打开飞书多维表格
2. 查看浏览器地址栏 URL
3. URL 格式：`https://feishu.cn/bitables/base/<APP_TOKEN>`
4. 复制 `<APP_TOKEN>` 部分（通常是 `bascnXXXXXXXX` 或 `bttcnXXXXXXXX`）

### 3.2 获取 Table ID

1. 在多维表格页面，点击数据表名称（默认"数据表 1"）
2. 点击「更多操作」→「查看表信息」
3. 复制 Table ID（通常是 `tblXXXXXXXX` 格式）

### 3.3 获取 API Token

#### 方式一：使用飞书开放平台（推荐）

1. 访问 https://open.feishu.cn/
2. 登录企业账号
3. 进入「企业自建应用」→「创建应用」
4. 应用名称：`考试系统`
5. 创建完成后，进入「凭证与基础信息」
6. 复制 `App Secret`

7. 进入「权限管理」
8. 添加以下权限：
   - `bitable:app` - 访问多维表格
   - `bitable:table` - 访问数据表
   - `bitable:record` - 读写记录

9. 进入「版本管理与发布」→ 发布应用
10. 使用以下 URL 获取 Tenant Access Token：

```bash
curl -X POST https://open.feishu.cn/open-apis/auth/v3/tenant_access_token/internal \
  -H "Content-Type: application/json" \
  -d '{
    "app_id": "cli_XXXXXXXX",
    "app_secret": "XXXXXXXX"
  }'
```

返回的 `tenant_access_token` 就是 API Token

#### 方式二：使用个人访问令牌（简单测试）

1. 访问 https://open.feishu.cn/document/ukTMukTMukTM/ukTMukTMukTM/ugjM14COiYjL0ITN
2. 点击「获取 Token」
3. 选择「个人访问令牌」
4. 生成并复制 Token

**注意：** 个人访问令牌有效期较短，适合测试使用

## 第四步：配置考试系统

打开 `index.html` 文件，找到第 267 行左右的配置：

```javascript
var FEISHU_CONFIG = {
  webhookUrl: '',     // 可选：飞书群机器人 Webhook
  appToken: '',       // 填入第三步获取的 App Token
  tableId: '',        // 填入第三步获取的 Table ID
  apiToken: ''        // 填入第三步获取的 API Token
};
```

填写你的配置信息：

```javascript
var FEISHU_CONFIG = {
  webhookUrl: 'https://open.feishu.cn/open-apis/bot/v2/hook/XXXXXXXX',
  appToken: 'bascnXXXXXXXXXXXXX',
  tableId: 'tblXXXXXXXXXXXXX',
  apiToken: 'uXXXXXXXXXXXXX'
};
```

## 第五步：配置群通知（可选）

如果希望考试成绩发送到飞书群：

1. 在飞书群中添加「机器人」
2. 复制 Webhook 地址
3. 填入 `webhookUrl` 配置

## 第六步：测试

1. 将修改后的 `index.html` 推送到 GitHub
2. 访问考试系统完成一次考试
3. 检查飞书多维表格是否有数据
4. 检查飞书群是否收到通知（如果配置了 webhook）

## 常见问题

### Q: API Token 过期了怎么办？
A: Tenant Access Token 有效期为 2 小时，需要定期刷新。建议使用以下方式之一：
- 使用个人访问令牌（有效期较长）
- 搭建一个简单的后端服务来管理 Token

### Q: 提交失败，控制台显示 403 错误
A: 检查应用权限是否正确配置，确保有以下权限：
- `bitable:app`
- `bitable:table` 
- `bitable:record`

### Q: 字段名称可以自定义吗？
A: 可以。修改 `submitToFeishu()` 函数中的 `fields` 对象，确保与多维表格中的字段名称一致。

### Q: 可以添加更多字段吗？
A: 可以。在多维表格中添加字段后，在 `fields` 对象中添加对应的键值对即可。

## 安全提示

⚠️ **重要：** API Token 具有访问你飞书数据的权限，请勿泄露！

- 不要将 Token 提交到公开的 GitHub 仓库
- 建议使用环境变量或后端服务管理 Token
- 定期更新 Token
